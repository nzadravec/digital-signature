# Napredni operacijski sustavi - laboratorijska vježba (FER)

Zadatak:
Ostvariti programski sustav za izradu i čitanje digitalne omotnice te izradu i provjeru digitalnog potpisa i digitalnog pečata koristeći već gotove, slobodno dostupne kriptografske algoritme. Od kriptografskih algoritama treba korisniku omogućiti izbor svih navedenih algoritama iz svake kategorije:

<ul>
<li>Simetrični kriptosustav: AES i 3-DES. Ponuditi na izbor sve moguće veličine ključeva za svaki algoritam te najmanje dva načina kriptiranja (ECB, CBC, OFB, CFB, CTR, ...).</li>
<li>Asimetrični kriptosustav: RSA, ponuditi nekoliko različitih veličina ključeva.</li>
<li>Funkcija za izračunavanje sažetka poruke (hash funkcija): SHA-2 ili SHA-3. Omogućiti izbor između najmanje dvije od četiri inačice algoritma, npr. SHA3-256 i SHA3-512).</li>
</ul>
